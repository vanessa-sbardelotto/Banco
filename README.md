### Sistema Bancário!
O projeto desenvolvido tem a intenção de simular um sistema bancário simples utilizando
Programação Orientada a Objetos. Nesse cenário foram criadas as classes Cliente e Conta,
que se relacionam por composição, visto que cada conta pertence a um client. De acordo com 
o que se pede no enunciado do trabalho, foi criada a classe Banco na qual funciona como 
controladora, gerenciando a lista de contas criadas. Também foi implementada a exceção personalizada
por meio da classe SaldoInsuficienteException, utilizada para impedir erros como por exemplo um 
saque acima do limite de saldo disponível. No menu principal, foi utilizado três blocos try/catch
que tratam erros de sistema. O programa oferece um menu interativo no console, permitindo criar 
contas, listar contas, realizar depósitos e efetuar saques com segurança e mensagens claras ao usuário.
